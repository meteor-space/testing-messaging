# Space Testing [![Circle CI](https://circleci.com/gh/meteor-space/testing-messaging.svg?style=svg)](https://circleci.com/gh/meteor-space/testing-messaging)

Helpers for Testing Space Messaging Infrastructure, implied by space:messaging.

## Install
`meteor add space:testing-messaging`

## Run the tests
`meteor test-packages ./`
